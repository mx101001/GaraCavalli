# Readme
## Gara Cavalli
### App java costruita per comprendere il funzionamento e l uso nel mondo reale dei thread
L'app permette di instaziare una corsa simulata nella quale randomicamente uno degli X cavalli viene azzoppati (si ferma) in fine l' app stampa la classifica della gara e salva la classifica in un file che viene scelto dall utente al momento del salvataggio
